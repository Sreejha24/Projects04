How to setup
==================
* Create a virtual directory in your IIS and point to this folder.
* Run it

For .NET Developer
==============
In Visual Web Developer or Visual Studio - Go to File > Open Web Site ... and open this folder and run.

All the samples url provided in the ebook is using 9007 port, you can setup this port number in the Project Property from Visual Studio
